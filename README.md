# DitElforbrugExcelPriser
Dit elforbrug fra din timeaflæste elmåler direkte forbundet til Excel. Og så endda med priser inkluderet!

Dette er en Excel-løsning, der trækker både forbrugsdata, stamoplysninger, elbørspriser og tariffer for at skabe et overblik over en husstands elforbrug time for time.
Se guide til anvendelse i arket "Om".
